# ⏳ The Infinite Moment

### "Every second is a chance to start fresh."

A professional C++ digital clock designed for the terminal. This project is a reminder that we are defined by our present choices, not our past judgments.

## ✨ Key Features
* **Live Updates:** Real-time hours, minutes, and seconds.
* **Aesthetic Design:** Clean ASCII frame for a better terminal experience.
* **Philosophical Touch:** Integrated quotes about time and renewal.

## 🚀 How to Run
1. Compile the file: `g++ infinite_moment.cpp -o clock`
2. Run the application: `./clock`
